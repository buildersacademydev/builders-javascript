# Builders JavaScript

JavaScript challenges for MERN Stack Bootcamp students.

## Setup

```bash
# Clone the repository
git clone https://github.com/buildersacademydev/builders-javascript
cd builders-javascript

# Install dependencies
npm install
```

## Running Tests

```bash
# Run all tests
npm test

# Run tests in watch mode
npm run test:watch

# Run specific level tests
npm run test:beginner
npm run test:intermediate
npm run test:advanced
```

## Structure

```
js-01/           # Day 1 - JavaScript Fundamentals
├── beginner/    # Single concept practice
├── intermediate/# Multi-concept challenges
└── advanced/    # Real-world scenarios
```

## How to Complete Challenges

1. Navigate to a challenge file in `challenges/` folder
2. Read the function description and TODO comments
3. Implement the function
4. Run the corresponding test file to verify your solution
5. All tests should pass before moving to the next challenge

## Topics Covered (Day 1)

- console.log
- Variable declaration (var, let, const)
- Primitive data types (String, Number, Boolean, Undefined, Null)
- Type coercion
- Equality operators (== vs ===)
- Functions
- if/else statements
- Loops (for, while, do-while)
