# JS-01: JavaScript Fundamentals

## Learning Objectives

By the end of this session, you will be able to:

1. Use `console.log` for output and debugging
2. Declare variables using `var`, `let`, and `const`
3. Identify and work with primitive data types
4. Understand and apply type coercion
5. Differentiate between `==` and `===`
6. Create and use functions
7. Write conditional statements
8. Implement loops for iteration
